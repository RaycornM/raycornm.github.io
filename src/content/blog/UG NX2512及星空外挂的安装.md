### UG安装

1. 使用CAP工具箱安装NX2512或者UE5.NET工具站安装

```bash
https://www.xu5.net/windows/industrial-engineering/496/
```

#### 安装教程

1.下载https://www.xu5.net/windows/industrial-engineering/496/ 完成安装包，解压缩。

![步骤1](https://www.xu5.net/wp-content/uploads/2026/03/1-3.png)

打开安装包中【补丁】文件夹中【programData】文件夹粘贴到C盘根目录

![步骤2](https://www.xu5.net/wp-content/uploads/2026/03/2-3.png)

粘贴到C盘根目录

![步骤3](https://www.xu5.net/wp-content/uploads/2026/03/3-3.png)fig.3

返回安装包【补丁】文件夹，运行【Siemens\_Licensing\_SSQ】注册表

![步骤4](https://www.xu5.net/wp-content/uploads/2026/03/4-3.png)fig.4

打开【安装文件夹】，运行【Launch】，开始安装

![步骤5](https://www.xu5.net/wp-content/uploads/2026/03/5-3.png)fig.5

点击【install Designcenter NX】开始安装

![步骤6](https://www.xu5.net/wp-content/uploads/2026/03/6-2.png)

选择中文

![步骤7](https://www.xu5.net/wp-content/uploads/2026/03/7-2.png)fig.7

下一步

![步骤8](https://www.xu5.net/wp-content/uploads/2026/03/8-2.png)fig.8

选择安装位置，不能用中文目录，以及选择安装产品（注意：安装位置不要安装到中文及特殊字符目录）

![步骤9](https://www.xu5.net/wp-content/uploads/2026/03/9-1.png)fig.9

这里许可证文件填写【C:\\ProgramData\\Siemens\\siemens\_SSQ.dat】（不要复制任何 空格）

![步骤10](https://www.xu5.net/wp-content/uploads/2026/03/10-1.png)fig.10

选择中文，下一步

![步骤11](https://www.xu5.net/wp-content/uploads/2026/03/11-1.png)fig.11

开始安装

![步骤12](https://www.xu5.net/wp-content/uploads/2026/03/12-1.png)fig.12

安装完成关闭所有窗口

![步骤13](https://www.xu5.net/wp-content/uploads/2026/03/13-1.png)fig.13

从开始菜单中找到【Designcenter NX】右键选择打开文件所在位置，就到了NX的安装位置。

![步骤14](https://www.xu5.net/wp-content/uploads/2026/03/14-1.png)f

然后点击顶部目录，返回【DesigncenterNX2512】文件夹

![步骤15](https://www.xu5.net/wp-content/uploads/2026/03/15-1.png)

然后就到了【DesigncenterNX2512】目录

![步骤16](https://www.xu5.net/wp-content/uploads/2026/03/16-1.png)

然后打开安装包，【补丁】文件夹中的【DesigncenterNX2512】，复制其中的所有文件

![步骤17](https://www.xu5.net/wp-content/uploads/2026/03/17-1.png)

然后粘贴到安装目录下面的【DesigncenterNX2512】，点击空白处粘贴，不要粘贴到某个文件夹去了。

![步骤18](https://www.xu5.net/wp-content/uploads/2026/03/18-1.png)

选择全部替换即可

![步骤19](https://www.xu5.net/wp-content/uploads/2026/03/19-1.png)fig.19

然后就可以使用了

![步骤20](https://www.xu5.net/wp-content/uploads/2026/03/20-1.png)

### 外挂安装

2. 安装星空外挂（以管理员身份安装，安装后重启计算机）

```bash
通过网盘分享的文件：星空8.0.zip
链接: https://pan.baidu.com/s/1cobIwwenwc0tDzI9fs_LLg?pwd=abcd 提取码: abcd
```

3. 打开CAP工具箱--点击外挂菜单打开安装路径（看看外挂是不是安装到2406路径里了）
4. 右键Siemens\NX2506\UGII\menus（ug安装路径）文件夹里的custom\_dirs.dat把只读勾选去掉，然后用记事本打开（如果不让编辑的话，可以先将文件粘贴到桌面进行编辑，编辑好后再剪切到源目录里），打开文件后，在文件的结尾追加路径：D:\\外挂安装路径\QuickCAM\\NX10.0
