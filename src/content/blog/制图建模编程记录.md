6. 加工攻丝注意：

   M6x1：

   创建丝锥

   ```
   创建刀具--hole-making--TAP--确定
   直径：6
   螺距：1
   其余默认
   ```

   创建工序

   ```
   插入hole_making工序--丝锥--刀具TAP6x1--确定
   转速：100RPM
   进给：如果单位是每分钟进给mmpm，那么就填写转速x螺距的值（如m5*0.8的丝锥转速设置为300，那进给率就是300*0.8=240，后处理出来就是F240.），如果单位是每转进给mmpr，那么就直接填写螺距值（需后处理支持）

   指定特征几何体：
   选择对象：选择孔特征或者圆曲线
   螺纹尺寸：通过修改长度值来设置螺纹的长度
   ```

   ![image](https://gh-proxy.org/https://github.com/RaycornM/person-picture-bed/blob/main/img/0b0802e5b0ceaa32dfd6130b7f2414c7.jpg)

   单位是每转进给mmpr，那么就直接填写螺距值，后处理在画圈位置修改为丝攻螺距，可以***==ctrl+f==***搜索M29字段快速找到攻丝代码

   > [!WARNING]
   > 同时如果攻牙深度较深，需要加一个反转代码防止断丝锥
   >
   > 如上图F1. 后方添加Q5. 表示每攻5mm深反转一次，根据丝锥大小和深度决定反转数值
   >

   ![](https://gh-proxy.org/https://github.com/RaycornM/person-picture-bed/blob/main/img/75e101797928bf645ded0eba4dfe8071.jpg)

   m5丝锥 0.8牙距

   ![](https://gh-proxy.org/https://github.com/RaycornM/person-picture-bed/blob/main/img/20260519143935091.png)

   进入后关闭图纸页
